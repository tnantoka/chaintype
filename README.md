# Chaintype

A small typing game inspired by hanitype, made with ebitengine.

https://chaintype.pages.dev/

![](/docs/chaintype.gif)

## Development

```
# Format
$ go fmt ./...

# Serve
$ go install github.com/hajimehoshi/wasmserve@latest
$ wasmserve
$ open http://localhost:8080/ 
$ curl localhost:8080/_notify
```
